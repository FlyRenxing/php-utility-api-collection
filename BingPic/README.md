# 获取必应 Bing 每日一图API

## 简介

此API可获得必应每日一图。GET接收图片分辨率、N天两个参数，API返回对应图片地址的302响应。

## 使用方式

`index.php?size=&daysago=`

| 参数名  | 默认值    | 说明                                     |
| ------- | --------- | ---------------------------------------- |
| size    | 1920x1080 | 获取指定大小的图片，分辨率有限制。见下文 |
| daysago | 0         | 获取N天前的图片，0为当天                 |
